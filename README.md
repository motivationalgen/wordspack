# Wordspack

Wordspack is a suite of free, fast, mobile-first online word tools designed for writers, students, gamers, and creators. It provides essential utilities in a clean, distraction-free interface.

## Key Features
- **Word Scrambler & Unscrambler**: Solve anagrams and find words for Scrabble and Wordle.
- **Word Counter**: Real-time word and character counting with reading time estimates.
- **Word of the Day**: Build your vocabulary with daily curated words and examples.
- **Typing Speed Test**: Measure your WPM and accuracy on diverse passages.
- **Random Word Generator**: Spark creativity with nouns, verbs, and adjectives.
- **Brand Name Generator**: Brainstorm catchy names for startups and projects.

## Tech Stack
- **Framework**: React + Vite
- **Styling**: Tailwind CSS
- **Components**: Radix UI (Accordion, etc.)
- **Icons**: Lucide React
- **SEO**: React Helmet Async + Structured Data (JSON-LD)

## Privacy
All processing happens entirely in your browser. No text is ever sent to our servers.

## Development
```bash
npm install
npm run dev
```
